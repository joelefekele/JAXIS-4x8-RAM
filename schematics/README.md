# Schematics

This folder contains the complete electrical schematics for the JAXIS 4×8 RAM.

The schematics document every hardware connection used throughout the project, from the initial block diagrams to the final circuit implementation.

The objective is to provide sufficient information for another engineer to understand, reproduce, and verify the design.

---

## Contents

This folder will include:

- System Block Diagram
- Register Module
- Address Decoder
- Write Logic
- Data Bus
- Read Logic
- Complete RAM Schematic
- PCB Schematics (Future)

---

## Design Philosophy

Every schematic in this project is drawn from first principles.

Rather than copying an existing RAM design, each subsystem has been developed independently before integration into the complete memory architecture.

The schematics are intended to explain both **how** the hardware is connected and **why** each connection exists.
