# Documentation

This folder contains the complete engineering documentation for the JAXIS 4×8 RAM project.

The documentation records every stage of the project's development, from the initial design concepts through hardware implementation, testing, and validation.

## Documents

- **ProjectOverview.md** — Project objectives, motivation, and expected outcomes.
- **SystemArchitecture.md** — Block diagrams and overall RAM architecture.
- **Theory.md** — Digital logic concepts used throughout the project.
- **DesignDecisions.md** — Engineering decisions and trade-offs.
- **TestingPlan.md** — Hardware testing procedures and validation methods.
- **LessonsLearned.md** — Challenges encountered and engineering lessons learned.

The objective of this documentation is to provide a complete engineering record of the project and demonstrate the design process behind the hardware.
